## Day 2: Dive!
### Part 1
```raku
say .<forward>.sum * (.<down>.sum - .<up>.sum) given lines».words.classify(*[0], as => *[1])
```

### Part 2
```raku
my ($h, $d, $a);
for slurp.words -> $k, $v {
  given $k {
    when ‘down’    { $a += $v }
    when ‘up’      { $a -= $v }
    when ‘forward’ { $h += $v; $d += $a * $v }
  }
}
say $h * $d
```
